# v0.9.3 Doctor JSON output

## Summary

This patch release adds JSON output support for `trip doctor`.

## Added

- Added `--json` output support to `trip doctor`.
- Added README documentation for doctor JSON output.
- Added tests for doctor JSON output.

## Notes

- JSON output is intended for tooling and automation.
- When `--json` is specified, stdout contains JSON only.
- Doctor JSON reflects the current diagnostic structure and is treated as an internal format for now.
