## Trip Doctor 検証

各チェックパターンの実出力は [`samples/trip_doctor/outputs/`](../../samples/trip_doctor/outputs/) に保存しています。

再生成:

```bash
bash samples/trip_doctor/generate_outputs.sh
```

| ファイル | 検証内容 |
|---|---|
| `01-clean-trip.txt` | 問題なし（`No major issues found.`） |
| `02-empty-itinerary.txt` | itinerary 0件でもエラーにならない |
| `03-overloaded-day.txt` | 1日7件以上の予定 |
| `04-no-restaurant.txt` | 食事（`restaurant`）カテゴリがない日 |
| `05-high-travel-time.txt` | 1日の移動時間合計 180分以上 |
| `06-missing-duration.txt` | 所要時間未設定の itinerary |
| `07-combined-issues.txt` | 複数警告が同時に出るケース |
