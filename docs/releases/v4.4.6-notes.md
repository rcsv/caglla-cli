# v4.4.6 — Travel Book presentation helpers extraction Phase 3

## Summary

Phase 3 refactor: note heading labels, itinerary note context, day heading labels, and trip date range moved from `markdown.rs` to `travel_book_presentation.rs`.

**No output behavior change** — Okinawa golden body unchanged except Colophon `Version:`.

## New helpers

- `travel_book_trip_date_range` — cover / colophon date range
- `travel_book_note_heading_label` — Note scope labels without Markdown syntax
- `travel_book_note_itinerary_context` — `Day N / time title` context for itinerary notes
- Daily schedule day headings reuse `travel_book_day_overview_label`

## Non-goals

- `TravelBookDocument` / full presentation model
- Markdown display rule changes
- Schema changes
