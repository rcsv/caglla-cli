# v0.9.2 Checklist JSON output

## Summary

This patch release adds JSON output support for checklist read commands.

## Added

- Added `--json` output support to `checklist list`.
- Added `--json` output support to `checklist show`.
- Added README examples for checklist JSON output.

## Notes

- JSON output is intended for tooling and automation.
- When `--json` is specified, stdout contains JSON only.
