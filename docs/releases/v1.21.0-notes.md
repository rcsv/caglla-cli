## Review conclusion

```text
Note is not limited to facts.
Note is an Annotation Layer — supplementary information attached to Trip, Day, or Itinerary.
It includes Fact, Observation, Decision, and Reminder, and does not carry narrative.
```

## Travel Ledger Responsibility Model (v1 final form)

```text
Note           = Annotation Layer
Summary        = Abstract Layer
Travel Journal = Story Layer
```

## Reviewed (documented)

### Narrative boundary

| Layer | Narrative | Examples |
|---|---|---|
| **Note** | No | Facts, observations, decisions, reminders |
| **Travel Journal** (future) | Yes | Experiences, emotions, stories |

### v1.20 refinement

v1.20 used `Fact` as shorthand for Note. v1.21 refines to **Annotation** without contradicting v1.20's Abstract / Journal separation.

**v1.20 document is not overwritten** — v1.21 adds precision.
