# v1.4.1 trip diff Note support

## Summary

- **`trip diff`** now compares `notes[]` from Export Schema v2
- Detects Note added / removed / changed with consistent display format
- v1 export (no `notes`) vs v2 export (`notes: []`) compares safely

## Added

- **Note diff** in `trip diff` — Trip / Day / Itinerary Note comparison
- **Display lines** — `+ Note added`, `- Note removed`, `~ Note changed`
- **Tests** — add/remove/body change for each Note type; v1 vs v2 empty notes; itinerary note title change

## Improved

- **`print_trip_diff`** — Notes section always shown; Itinerary early-return removed

## Notes / Compatibility

| Topic | v1.4.1 behavior |
|---|---|
| **Export schema** | Unchanged (still v2) |
| **Note diff keys** | Trip: title; Day: day_number + title; Itinerary: day_number + sort_order + itinerary title |

## Breaking Changes

None.

## Verification

- `make check`
- Note add/remove/change detected by `trip diff`
- v1 export comparison does not panic
