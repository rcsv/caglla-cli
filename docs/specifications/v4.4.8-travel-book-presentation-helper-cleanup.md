# v4.4.8 — Travel Book presentation helper cleanup (implementation)

v4.4.7 final review で残っていた presentation helper 内の Markdown 構文混入 2 件を解消する **golden 不変の small cleanup**。

| Item | Status |
|---|---|
| Release | v4.4.8 |
| Scope | Markdown syntax boundary fix only — no output behavior change |
| Validation fixture | [Okinawa Sesoko 2026 sample](../../samples/okinawa_sesoko_2026/README.md) |

## Goal

Presentation helper は renderer-independent な label / value / context を返し、Markdown 構文（bullet / bold）は `markdown.rs` が付与する。

## Changes

### 1. Category detail

| Before | After |
|---|---|
| `format_travel_book_category_detail_line` → `"- 種別: 移動"` | `travel_book_category_detail_label` → `("種別", "移動")` |
| bullet `- ` in presentation | `markdown.rs`: `format!("- {label}: {value}")` |

### 2. Reservation heading

| Before | After |
|---|---|
| `format_travel_book_reservation_heading` → `"**Day N / …**"` | `travel_book_reservation_heading` → `TravelBookReservationHeading { main_label, provider_suffix }` |
| bold `**` in presentation | `markdown.rs`: `format_reservation_heading_markdown` |

## Non-goals

- `TravelBookDocument` / `TravelBookItem` introduction
- Markdown output spec change
- DB / JSON export schema change
- Venue / Maps / GUI / schedule role
