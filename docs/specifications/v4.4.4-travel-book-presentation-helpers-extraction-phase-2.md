# v4.4.4 — Travel Book presentation helpers extraction Phase 2 (implementation)

Phase 2 extraction from the v4.4.2 helper review §13.

| Item | Status |
|---|---|
| Release | v4.4.4 |
| Scope | Module move / helper extraction only — no output behavior change |
| Validation fixture | [Okinawa Sesoko 2026 sample](../../samples/okinawa_sesoko_2026/README.md) |

## Goal

Move remaining presentation decisions from `markdown.rs` into `travel_book_presentation.rs`.

## New helpers

| Function | Role |
|---|---|
| `trip_overview_worth_showing` | Trip overview chapter visibility |
| `trip_overview_time_metrics_worth_showing` | Omit Stay/Travel/Total when all zero |
| `collect_days_overview_entries` | Days overview — non-empty summaries only, sorted |
| `travel_book_day_overview_label` | `Day N — date` label |
| `planned_cost_note_column_visible` | Note column when any estimate note non-empty |
| `planned_cost_estimate_display_title` | Empty title → `-` |
| `planned_cost_itinerary_group_label` | `Day N / time title` group context |

## Stays in `markdown.rs`

Markdown renderer concerns: headings, tables, bullets, blank lines, bold.

## Non-goals

- `TravelBookDocument` abstraction
- Output / golden body changes (Colophon `Version:` only)
- DB / JSON export schema changes
