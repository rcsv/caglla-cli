# v4.4.3 — Travel Book presentation helpers extraction (implementation)

Phase 1 extraction from the v4.4.2 helper review.

| Item | Status |
|---|---|
| Release | v4.4.3 |
| Scope | Module move only — no output behavior change |
| Validation fixture | [Okinawa Sesoko 2026 sample](../../samples/okinawa_sesoko_2026/README.md) |

## Goal

Move existing Travel Book presentation helpers from `src/io/markdown.rs` into `src/io/travel_book_presentation.rs`.

## Moved functions

| Function | Role |
|---|---|
| `format_travel_book_category_detail_line` | Category display name line |
| `travel_book_note_sort_key` | Notes sort key (Trip → Day → Itinerary) |
| `sort_export_notes_for_travel_book` | Notes sort |
| `reservation_provider_line_redundant` | Provider subtitle dedup |
| `format_travel_book_reservation_period` | Human-readable reservation period |
| `format_travel_book_reservation_heading` | Reservation heading with optional provider |

Private helpers (`parse_reservation_datetime`, etc.) move with reservation period logic.

## Module layout

```text
src/io/travel_book_presentation.rs  — presentation rules (renderer-independent)
src/io/markdown.rs                  — Markdown renderer (calls presentation module)
```

## Non-goals

- `TravelBookDocument` abstraction
- Output / golden body changes (Colophon `Version:` only for cargo bump)
- DB / JSON export schema changes
